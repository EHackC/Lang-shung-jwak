# LANG-SHUNG-JWAK (랭슝좍)

랭슝좍은 트릭컬 리바이브의 리슝좍 밈에서 착안하여, 코드를 짜면서도 비비와 순수를 따잇하는 재미를 느끼기 위해 만들어진 언어입니다. 현재 파이썬으로 구현되어 있습니다.

엄랭 Github에서 아이디어를 얻어 제작했습니다. [[엄랭 Github](https://github.com/rycont/umjunsik-lang)]

## 개요

이 언어를 만들면서 고려한 부분은 다음과 같습니다.

- 기본적인 프로그래밍 언어의 역할을 할 것.
- 코드를 읽었을 때, 실제 리슝좍이 말하는 듯한 느낌을 줄 것.
- 코드 작성자의 재량에 따라 같은 명령어에 대해서도 다양한 형태로 구현가능하여 작성자의 다양한 감정을 드러낼 수 있도록 할 것.

코드 작성 TIP: 코드 작성 시 코드의 내용을 마치 리슝좍 본인인 양 구두로 말하면서 작성해 보세요. 좀 더 풍부한 감정을 코드에 담을 수 있습니다.


## 문법

랭슝좍에서 사용하는 키워드는 다음과 같습니다. 줄 단위로 명령어가 구성됩니다.

- 슝, 슈웅, 슈우웅, ...
- 좍, 좌악, 좌아악, ...
- 비비 ... 따잇
- 순수 ... 따잇
- ,
- ~
- @
- ;

아울러 표현의 자유도를 높이고 감정을 풍부하게 전달하기 위해 다음의 문자는 명령어 중간에 사용할 수 있습니다.
- ?
- !
- .
- " " (공백)

### 교주님

랭슝좍 언어는 반드시 시작 시 '교주님'을 사용합니다.
```python
교주님

# 이후 코드...
```

### 자료형

현재는 정수형만 지원되며, '좍', '좌악', '좌아악', ...을 이용합니다.

```python
좍          # 1
좌악        # 2
좌아악      # 3
좌아아악    # 4
# 이하 '아'가 추가될 때마다 1씩 증가
```

### 연산
자료형-자료형, 변수-변수, 변수-자료형 사이에 사용할 수 있습니다. 덧셈 '~', 뺄셈 ';', 곱셈 ',', 나눗셈 '@'을 이용합니다. 연산은 우선 순위 없이 무조건 왼쪽부터 진행됩니다.
```python
좍~좍           # 1+1
좌악,좌아악     # 2x3
슝;좌아악       # 첫번째 변수에 저장된 값 - 3
슈웅;슈우웅@좍  # (두번째 변수에 저장된 값 - 세번째 변수에 저장된 값) / 1
```


### 변수 선언

문장의 맨 앞에 '슝', '슈웅', '슈우웅', ...을 사용하여 변수를 선언합니다. 변수 선언시에는 다른 변수를 대입하거나 정수형 자료형을 이용하여 값을 저장해야 합니다.
```python
슝좍                    # 첫번째 변수에 1을 대입.
슝...?좍!!!            # 첫번째 변수에 1을 대입. (표현의 자유도)
슈웅좌아악,좌악,좌악     # 두번째 변수에 3x2x2=12를 대입.
슈우웅슈웅~슝           # 세번째 변수에 두번째 변수의 값 + 첫번째 변수의 값 = 7을 대입.
```


### 출력
#### 변수에 저장된 값에 해당하는 ASCII 코드값을 출력하고 싶은 경우
'비비'로 문장을 시작하고, 이후 '따잇'을 이용하여 변수에 저장된 값을 출력할 수 있습니다. 'ㅋ'의 개수가 변수 인덱스를 의미합니다. 'ㅋ'의 위치는 자유롭게 둘 수 있습니다. 단 '비비', '따잇' 내에 위치해서는 안됩니다.
```python
비비따잇        # 첫 번째 변수의 값 출력
비비따잇ㅋㅋ    # 세 번째 변수의 값 출력
비비ㅋ따잇ㅋ    # 세 번째 변수의 값 출력
비비ㅋㅋㅋ따잇  # 네 번째 변수의 값 출력
비ㅋㅋ비따잇    # SyntexError
비비따ㅋㅋㅋ잇  # SyntexError
```
#### 변수에 저장된 값을 출력하고 싶은 경우
'순수'로 문장을 시작합니다. 나머지 부분은 위와 동일합니다.
```python
순수따잇        # 첫 번째 변수의 값 출력
순수따잇ㅋㅋ    # 세 번째 변수의 값 출력
순수ㅋ따잇ㅋ    # 세 번째 변수의 값 출력
순수ㅋㅋㅋ따잇  # 네 번째 변수의 값 출력
```
